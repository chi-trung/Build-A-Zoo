# 🎂 Build A Zoo Script

Script hỗ trợ cho game **Build A Zoo** trên Roblox — tương thích với executor **Xeno**.

## 🚀 Cách sử dụng

Mở executor (Xeno) → Inject → paste dòng bên dưới → **Execute**:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/chi-trung/Build-A-Zoo/main/BuildAZoo.lua"))()
```

Menu sẽ hiện ra. Bấm nút **EN** ở góc phải để đổi ngôn ngữ **English / Tiếng Việt**.

## ✨ Chức năng

### Main (Chính)
- Auto Claim Coins — tự nhặt tiền của **tất cả** pet (cả pet vườn lẫn Big Pet)
- Claim Delay
- Coins X2 / Anti Lag
- Auto Lotto, Auto Conveyor
- No Client (ẩn người chơi khác cho đỡ lag)
- Dino Tasks, Auto Claim Season Pass

### Fishing (Câu Cá)
- Auto Fish / Water Method + delay
- Infinite Auto Fish (any position)
- Instant Skip, Full Auto Fish

### Movements (Di Chuyển)
- Anti AFK, Infinite Jump
- Walk Speed, Noclip
- Gravity, Jump Power, Field of View

### Settings (Cài Đặt)
- Minimize Keybind (`RightControl`)
- UI Blur, UI Transparency

## 📝 Lưu ý
- Raw GitHub có thể cache ~5 phút, nên sau khi cập nhật script hãy đợi một chút để lấy bản mới.
- Script chỉ dùng cho mục đích học tập / giải trí.
